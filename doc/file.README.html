ruby-upworthy

A clickbait title generator, translated into Ruby from Mike Lacher's Upworthy Generator.

Installation

$ gem install ruby-upworthy

Usage

From Ruby:

require 'upworthy'

Upworthy.headline # => "Watch an extroverted police officer become a legend with six words"

From the shell:

$ upworthy
